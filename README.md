# CAMP: Compiler and Allocator-based Heap Memory Protection

*Zhenpeng Lin, Zheng Yu, Ziyi Guo, Simone Campanoni, Peter Dinda, and Xinyu Xing*




## Prerequisite

- Ubuntu 22.04

- Clang 12.0.1

## Build

```bash
mkdir build && cd build
cmake .. && make
```
## Usage

```bash
tools/vcc <compile options>
tools/v++ <compile options>
```
